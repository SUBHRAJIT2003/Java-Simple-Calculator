# Simple Java Calculator

A beginner-friendly, console-based calculator built with Java. This project demonstrates foundational backend programming concepts including taking user input, conditional logic (`switch` statements), and basic error handling.

<--Features-->

* **Basic Arithmetic Operations:** Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
* **Interactive Console:** Prompts the user to enter numbers and operators step-by-step.
* **Error Handling:** Prevents program crashes by explicitly checking for and handling division by zero.

[] Prerequisites

To run this project locally, you will need:
* **Java Development Kit (JDK)** installed on your machine.
* A code editor or IDE (like VS Code, IntelliJ IDEA, or Eclipse).
